# MVP limitations and next steps

## Included in this build

- Map-first landing with 10 Partido municipality polygons (PSGC-sourced, enriched)
- Municipality selection, summary, profiles, place pages
- Authoritative v0.10 repository import (267 places, 56 photo records)
- Grounded AI concierge (rule-based; provider hook ready)
- Itinerary + budget estimate + local trip save
- Supabase schema + RLS foundation
- Competitive UX benchmark and research plan

## Not yet complete

- Full Supabase production import pipeline (runtime cache used for local dev)
- Admin dashboard UI
- Community contribution and validator workflows
- External LLM orchestrator and full multi-agent suite
- 360° panorama viewer
- Native mobile parity and offline cache
- Filipino and Bikol message catalogs
- Live usability testing (plan documented)

## Next phases

Follow MVP order: tourism directory depth → contributions → agentic AI expansion → virtual tours → mobile/offline.
